Iris Flower Classifier

Welcome to the Iris Prediction App! This application uses a machine learning model to predict the species of Iris flowers based on their sepal and petal dimensions.

About the Iris Dataset

The Iris flower dataset is a famous dataset in pattern recognition and is frequently used for testing out machine learning algorithms. It includes data from three different species of Iris (Iris setosa, Iris virginica, and Iris versicolor). Each sample in the dataset has four features: sepal length, sepal width, petal length, and petal width, measured in centimeters.

How to Use

To predict the Iris flower species, enter the dimensions of the flower's sepals and petals below, and click "Predict".
